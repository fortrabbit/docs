## About MySQL

MySQL is the M in the LAMP stack. It's an open-source relational database management system that uses SQL as its query language. It is one of the most popular database systems in the world and is widely used for web applications, CMS and other data-driven applications.

## Booking MySQL

MySQL is integrated as an optional component. It's available in different plans. Each environment can have one MySQL component containing one database booked. The database can has multiple tables.
