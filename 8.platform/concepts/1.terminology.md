## Key concepts

- [App](/app) - think git repo, groups environments and collaboration together
- [Environment](/environment) - think git branch, actual hosting is happening here
- [Account](/account) - access for a person to control the fortabbit hosting system
- [Payment method](/payment-method) - object holding payment credentials paying for apps

- **[App](app)**: a virtual instance for your PHP application
- **[Environment](environment)**: a virtual instance for your PHP application
- **Environment URL**: a unique default URL each App has: appname.frb.io
- **Account**: a Dashboard profile / login
- **Events**: a history log in the Dashboard
- **Payment method**: payment and billing group for Apps
- **Team**: [collaboration](collaboration.md) group
  - **Admin**: a [team member](collaboration.md) role within the Dashboard
  - **Owner**: a [team member](collaboration.md) role within the Dashboard
- **Component**: a service that can individually be booked and scaled (Professional App)
- **[Dashboard](dashboard)**: the master control panel: [dashboard.fortrabbit.com](https://dashboard.fortrabbit.com)
- **[Worker](worker-pro)**: an optional Component
- **PHP request**: a single call for a PHP file
