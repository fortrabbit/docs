## About Craft Copy

Craft Copy syncs your local development environment with your fortrabbit app environment â€” up and down. It conveniently deploys code changes and synchronizes latest images and database entries. As a Craft CMS plugin will be installed locally and on the fortrabbit app environment.

![Craft Copy demo](https://github.com/fortrabbit/craft-copy/raw/master/resources/craft-copy-db-up.gif)

### Under the hood

- code up / down - git wrapper for push/pull
- volumes up / down - rsync wrapper
- db up / down - mysql dump wrapper

## Installing and using Craft Copy

Please see the **[Craft Copy usage guide at GitHub](https://github.com/fortrabbit/craft-copy)**. It guides you through the process of installing the plugin, connecting your app, deploying code, assets and database.
