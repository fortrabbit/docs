## Get ready

To connect to the remote MySQL database, it first needs to be booked. The [MySQL component](/platform/components/mysql) is optional but pre-booked for database-driven software systems.

To be able to connect to the remote database from your computer, also a [local development environment](/4.integrations/01.local-development/01.intro.md) with `mysql` and `mysqldump` is required for terminal usage. A database GUI such as Workbench or Sequel Ace is another option. Different local development environments come with different paradigms for working with a database. For containerized systems, the database service might not be directly exposed.
