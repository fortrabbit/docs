| Date            | Action                                  |
| --------------- | --------------------------------------- |
| End of month    | Invoice creation for monthly usage      |
| 1st             | Invoice distribution for previous month |
| 3rd             | Credit card payment processing          |
| 7th             | SEPA direct debit execution             |
| Until month end | Review and retry of bounced invoices\*  |

\* See [late payments](/1.platform/12.billing/23.late-payments.md) for details
