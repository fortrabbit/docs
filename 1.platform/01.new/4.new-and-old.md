**The old platform** is still open and fully functional and will receive full support and is only going to be slowly phased out. There is no rush to leave. We aim to make the migration as seamless as possible. Details and timing to follow.

**The new platform** is now live and open for everyone to use. Experience a redesigned new yet familiar fortrabbit. Use the same e-mail address as your old account so we can map it.

## New customers

Just use the new platform.

## Existing customers

Use both platforms. **Accounts have not been transferred.** Create a new account on the new platform, best use the same email address, so that we can match them later on. Create new apps on the new platform, configure existing apps on the old platform.

## Timeline

By the time of this writing, it's not defined yet how long the old platform will exactly be supported. It depends on feedback and the needs of our customers.
