1. :ContentLink{text="Sign up" href="/signup" prefix="dash"} to create a personal account.
2. Explore the dashboard.
3. Check the [install guides](/2.guides/)
4. Create a [free trial](/1.platform/02.get-started/02.free-trial.md) app.
5. Access code [SSH](/1.platform/04.code-access/03.ssh.md) or [SFTP](/1.platform/04.code-access/04.sftp.md).
6. [Deploy](/1.platform/05.deployment/01.intro.md) with Git.
7. [Migrate existing projects](/3.dev/20.how-to/migrate-to-fortrabbit.md) or start new ones.
8. Invite developers or clients to [collaborate](/1.platform/20.collaboration/1.intro.md).
9. :ContactUs{text="Contact us"} if you are stuck somewhere.
10. Upgrade to paid if you like it.
