[Environments](/1.platform/10.objects/02.environment.md) ship with an opinionated (think optimized) pre-configuration of PHP settings. Those also depend on the [software template](/1.platform/03.concepts/05.software-templates.md). The dashboard exposes the most important settings to configure the PHP version and certain PHP extensions.

:BlockLink{title="Edit root path vars for {{app-name}} / {{app-env-name}}" path="/environments/{{app-env-id}}/php"}

## Available settings

- PHP version
  - Choose a specific version or pin to the latest
- PHP extensions
- PHP ini settings
  - `post_max_size`
  - `max_input_size`
  - `output_buffering`
  - `max_execution_time` ‚Üê Keep it low. [Why?](/3.dev/02.performance/06.php-processes.md)
  - `upload_execution_filesize`

## See all specific settings

Use [phpinfo](/3.dev/15.learn/10.php-info.md) to dump all the PHP settings.

## Add additional settings

Use [user.ini](/3.dev/20.how-to/userini.md) for more fine grained control over PHP settings.
