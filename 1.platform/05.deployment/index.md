You connect a GitHub repository through the fortrabbit GitHub app, pick a branch per environment, and every push runs your build commands, swaps the new release in, and runs any post-deploy commands. The pages below cover the moving parts in detail, from triggers and deploy hooks to cached directories and troubleshooting.
