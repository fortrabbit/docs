Run modern PHP websites and web applications, specifically [Craft CMS](/2.guides/3.craft/1.setup.md), [Laravel](/2.guides/2.laravel/1.setup.md), [Statamic](/2.guides/4.statamic/1.setup.md), [October CMS](/5.more/6.october-cms.md) and [WordPress](/5.more/5.wordpress.md) here. The [GitHub integration](/6.deployment/2.github-app.md) provides deployment directly from the repo. Projects are represented as [apps](/1.platform/10.objects/1.app.md) with [environments](/1.platform/10.objects/2.app-environment.md). [Collaboration features](//16.collaboration/1.intro.md) map real world relations between developers and stakeholders.
