This section is the cookbook: short, task-shaped pages that answer questions like how to [migrate an existing site to fortrabbit](/3.dev/20.how-to/migrate-to-fortrabbit.md), [reduce hosting costs](/3.dev/20.how-to/4.reduce-hosting-costs.md) on a slow project, [upgrade PHP](/3.dev/20.how-to/upgrade-php.md) on a live environment, or copy files around with [rsync](/3.dev/20.how-to/rsync.md) and [scp](/3.dev/20.how-to/scp.md).

Each page is self-contained, so jump straight to the recipe you need. If a workflow is missing, the [platform docs](/1.platform) and the [guides](/2.guides) usually cover the underlying primitives.
