## License model

Kirby CMS is open source, but not free software. You can test it for free. Support the authors and the longevity of the project by obtaining a license for production.

## Understand flat file system

Unlike [WordPress](/4.wordpress/) or [Craft CMS](/3.craft-cms/) Kirby is a file based CMS. So there is no database, the actual contents are markdown text files written on the file system. Other flat file systems are [Grav](/5.guides/7.grav.md) and [Statamic](/4.statamic/1.setup.md). In general, this design approach makes things more easy, but also has some gotchas.

## Deployment options

Special consideration should be taken when deploying Kirby CMS. We assume you have a [local development environment](/15.tips/local-development.md) running.

- [Git + rsync deployment](/4.kirby/2.git-and-rsync.md) - advanced, sophisticated, recommended
- [rsync sync](/4.kirby/3.rsync.md) - advanced, quick
- [SFTP upload](/4.kirby/4.sftp-upload.md) - easy peasy
