## Invoice number

Each invoice contains an individual number. This number contains links back to the payment method and the service period.

## Service period

The service period is printed on the invoice.

## Invoice recipient

An invoice recipient postal address is included with each invoice. The current values from the payment method are copied. 

## Invoice item rows

Each invoice contains a number of rows. They are grouped by app / environment as each has a set of booked components. Each invoice row contains the booked component and it's plan and how many days it is billed for.

### Invoice rows example

| Component plan | Days billed | Daily price |  Month |
| -------- | -------- | -------- | -------- |
| PHP M | 30 | $0.50 | $15.00 |
| PHP M | 30 | $0.50 | $15.00 |
| Traffic S | 30 | $0.03 | $1.00 |
