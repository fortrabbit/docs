::CallOut{alert}
Laravel Nightwatch is currently not supported.
::

Laravel Nightwatch is a new PHP profiling tool in the Laravel ecosystem. It requires a daemon process to run on a [worker job](/1.platform/09.components/07.jobs.md), but since that process needs to be reached from outside the app environment, which is currently not possible on fortrabbit, we cannot support it yet.

Let us know if you want it. Your signal helps us. :ContactUs{text="Get in touch"}
