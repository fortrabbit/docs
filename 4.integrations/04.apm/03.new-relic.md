## About New Relic

[New Relic](https://www.newrelic.com) is a suite of professional software analysis tools that provides analytics and deep insights into application performance. New Relic integrates as a PHP extension, which is installed (but not enabled) on all fortrabbit apps, helping you identify performance bottlenecks and runtime issues.

## Usage

fortrabbit supports a Bring-Your-Own-License (BYOL) integration with New Relic. To get started, you need a New Relic account, then paste your license key into the fortrabbit dashboard.

### Over at New Relic

1. Sign up or log in to [New Relic](https://newrelic.com).
1. Add a new application
1. Copy the license key

The New Relic dashboard interface changes frequently. Here's how to find your license key:

- No need to install anything on the server
  - My Profile > My API > Create new LICENSE key
  - Copy the generated key value

### At fortrabbit

1. Log in to the dashboard
1. Navigate to your environment > settings > New Relic
1. Paste the New Relic license key
1. Save

The setup is complete. After 5 minutes, data should appear in your New Relic dashboard. For more information about using New Relic, see their [APM documentation](https://docs.newrelic.com/docs/apm/new-relic-apm/getting-started/introduction-new-relic-apm).
