## GitHub

fortrabbit integrates natively with GitHub through a dedicated [GitHub App](/1.platform/05.deployment/02.github-app.md), providing deep platform integration and the recommended deployment method.

## Other providers

[GitLab](04.gitlab.md) and [Bitbucket](03.bitbucket.md) are also supported. Since there is no native integration, deployment requires setting up a CI/CD pipeline using GitLab CI or Bitbucket Pipelines to sync code to fortrabbit.

[Codeberg](05.codeberg.md) is available as well.
