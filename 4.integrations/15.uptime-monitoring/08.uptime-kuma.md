[Uptime Kuma](https://uptimekuma.org/) is a self hosted uptime monitoring service with a modern UI and broad check support. It is a common choice when you prefer to run monitoring infrastructure yourself.

- Project website: [uptimekuma.org](https://uptimekuma.org/)
- Source code: [louislam/uptime-kuma](https://github.com/louislam/uptime-kuma)
