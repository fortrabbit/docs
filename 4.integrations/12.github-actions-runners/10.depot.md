[Depot](https://depot.dev/) is aims to accelerate Docker image builds, but it also supports GitHub Actions workloads with managed remote builders, centralized caching, ARM support, macOS builds, and optional GPU-backed jobs.

If your CI time is dominated by container builds rather than generic test execution, Depot can be a better choice than a general-purpose runner service.
