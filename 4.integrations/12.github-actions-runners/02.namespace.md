[Namespace](https://namespace.so/) is a managed runner platform aimed at teams that want a near drop-in replacement for GitHub-hosted runners with better storage and caching behavior. It also offers ARM support, macOS runners, remote terminal access, and additional tooling around preview environments and remote Docker builds.

It is a sensible option when build performance depends heavily on cache locality or when workflows need more disk space than the default GitHub runners provide.
