It's made to deploy and deliver modern PHP websites and web applications, specifically Craft CMS, Laravel, Statamic, October CMS and WordPress. The GitHub integration provides deployment directly from the repo. Projects are represented as apps, optionally with multiple environments. The collaboration features map real world relations between developers and stakeholders.

fortrabbit is self funded with a long time commitment since 2012.
