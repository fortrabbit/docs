It's made to deploy and deliver modern PHP websites and web applications, specifically [Craft CMS](/2.craft/1.setup.md), [Laravel](/4.guides/4.laravel.md), [Statamic](/3.statamic/1.setup.md), [October CMS](/4.guides/6.october.md) and [WordPress](/4.guides/5.wordpress.md). The GitHub integration provides deployment directly from the repo. Projects are represented as apps, optionally with multiple environments. The collaboration features map real world relations between developers and stakeholders.

fortrabbit is [self funded with a long time commitment](https://www.fortrabbit.com/about-us) since 2012.
